Install and Configure Windows 10

During Install

  1. Insert the USB flash drive containing Windows 10 and boot from it.
  2. Go through the installation procedure.
  3. Choose the Windows 10 partition when prompted (this should be partition number 1). Note: Because we are telling Windows to install in a specific partition, it will not generate a separate System Reserved partition. Instead, it will place all files in the partition we have chosen.
  4. Make sure not to choose “Express Install” when prompted (otherwise, several invasive privacy settings will be enabled).
  5. When prompted for the first user account, choose the name admin but ensure it is created as a local account not a One Drive account.

After Install

  1. Disconnect from the internet.
  2. Boot into the Windows 10 environment you just installed.
  3. Install all the drivers previously downloaded, in the following order:
    • chipset
    • LAN
    • audio
    • USB 3
    • graphics card
  4. Create a local, non-adminstrator account called johndoe.
  5. Create a visitor account by running this batch script from an administrator command prompt. When the script prompts for a visitor password, just press <ENTER>.
  6. Enable restore points on the C partition (Windows 10 disables them by default).
  7. Windows 10, by default, assumes the hardware clock is in local time. Linux Mint, by default, assumes it is in UTC time. To reconcile this difference, we will perform a registry edit to tell Windows 10 that the hardware clock is in UTC time. Download the registry edit and run it as administrator.
  8. Delete the hidden file C:\Users\johndoe\Music\desktop.ini and map the Music partition to this folder.
  9. Create a folder C:\Users\johndoe\Google Drive and map the Google Drive partition to it.
  10. Create a folder C:\Users\johndoe\Shared Documents and map the Shared Documents partition to it.
  11. Create a folder C:\Users\johndoe\Ebooks and map the Ebooks partition to it.
  12. Create a folder C:\Users\johndoe\VirtualBox VMs and map the Windows Virtual Machines partition to it.
  13. Set permissions for the Data partition so that only admin and johndoe can access it.
  14. Connect to the internet and install your security software.
  15. Install all outstanding Windows 10 updates.
  16. Install and activate any non-free software which you previously deactivated (ie. Microsoft Office).
  17. Download and install the driver and software for your printer.

Finish Up Windows Installation

  1. Remove unnecessary programs from startup using Windows' msconfig utility.
  2. Configure the desktop as desired.
  3. Calibrate the display using Windows' calibrate utility.
  4. Disable Fast Startup from the Power Options section of Control Panel. (This is needed in order to boot from the USB stick containing Linux Mint.)
  5. Disable hibernation. From an Administrator’s Command Line prompt, enter the following: C:> powercfg /h off (This step is needed in order to mount NTFS partitions as read-write under Linux Mint.)
